Instructions

Instructions


1. Choose a harvesting tool from your tool box.

2. Each tool can be used to harvest specific elements in the game.

3. Harvested elements can be found in your inventory and can be used to creat surprizing new materials.

4. Use the reset button to start over.

5. Use the music buttons to control the volume of the background music.


Pickaxe: use to mine stone, gold, and silver.

Axe: use to break wood and trees, and bushes.

Shovel: use to pick up dirt, ground, snow, etc.

Hoe: use to harvest lave and bricks.


To summon the shortcut keyboard, please press K.

Have fun!

NOTE: This mini-game was developed in two days
